# BPRO Mobile Web App (React)

A premium **Mobile-Web Application** for the BPRO Soccer Training Platform. This version is designed to look and feel like a native app while being compatible with any web backend like **Laravel** or **Node.js**.

## 🚀 Concept
- **Mobile-First Design**: Built with Tailwind CSS for high responsiveness.
- **PWA Ready**: Can be installed on any phone via browser as an app icon.
- **Backend-Native**: Perfect to be used as a Frontend for **Laravel API**.

## 🎨 Tech Stack
- **React (Vite)** — Fast and modern frontend.
- **Tailwind CSS** — Custom premium styling.
- **Lucide React** — High-quality icons.
- **Framer Motion** — Smooth mobile transitions and animations.

## 🛠️ How to View Locally
Since this is a Web App, you don't need a heavy SDK. Check the code and run:

```bash
cd d:\flutter\bpro_mobile_web
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser. Use **Inspect Element -> Mobile View** for the best experience.
